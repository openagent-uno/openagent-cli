# CLAUDE.md

## Before making any change

Read OpenAgent's vision document first:
https://github.com/openagent-uno/openagent-docs/blob/main/vision.md

It is the source of truth for OpenAgent's long-term goals and project standards. When the implementation and vision.md diverge, vision.md is correct and the implementation is what needs to change. Design choices, refactors, new features, and bug-fix strategy in this repo must all derive from it.

If the `openagent-docs` repo is checked out as a sibling directory, read the local `vision.md` from there; otherwise fetch the URL above.
